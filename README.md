# 📘 Візуалізація даних сенсорного слайдера

> Програмний проєкт для побудови візуалізацій сенсорних даних, аналізу просторового розподілу сигналів, взаємозв’язків між сенсорними каналами та похибок прогнозування координат точки дотику.

## 📌 Загальна інформація

- **Тип проєкту**: Програмний скрипт для аналізу та візуалізації сенсорних даних
- **Мова програмування**: Python
- **Основні бібліотеки**: NumPy, Pandas, Matplotlib, Seaborn, SciPy, scikit-learn
- **Вхідні дані**: Excel-файли з координатами точки дотику та сигналами сенсорів
- **Призначення**: Контроль, діагностика та візуальний аналіз роботи лінійного сенсорного слайдера

---

## 🧠 Опис функціоналу

- 📥 Зчитування експериментальних даних із файлу `data_slider_1.xlsx`
- 📊 Побудова просторових карт сигналів окремих сенсорів
- 🟦 Побудова карти домінуючого сенсора для визначення зон відповідальності каналів
- 📉 PCA-представлення восьми сенсорних сигналів у двовимірному просторі
- 🔗 Побудова кореляційної матриці для аналізу взаємозв’язків між сенсорами
- 📍 Аналіз похибок прогнозування координат точки дотику
- 🌡️ Побудова теплової карти просторового розподілу похибок
- 📈 Побудова гістограми розподілу похибок

---

## 🧱 Опис основних файлів і функцій

| Файл / Функція | Призначення |
|---------------|-------------|
| `data_slider_1.xlsx` | Основний файл із координатами точки дотику `X`, `Y` та сигналами сенсорів `S1–S8` |
| `Err_data_slider_1.xlsx` | Файл із даними для аналізу похибок прогнозування координат |
| `plot_sensor_map(sensor_id)` | Будує просторову карту сигналу вибраного сенсора |
| `plot_dominant_sensor_map()` | Визначає сенсор із максимальним сигналом у кожній точці та будує карту домінування |
| `plot_pca()` | Виконує PCA-аналіз і відображає багатовимірні сенсорні сигнали у двовимірному просторі |
| `plot_correlation_matrix()` | Будує кореляційну матрицю між вісьмома сенсорними каналами |
| `plot_error_analysis(error_file)` | Аналізує похибки прогнозування координат і будує графік, гістограму та теплову карту похибок |

## ▶️ Як запустити проєкт "з нуля"

### 1. Встановлення Python

Для роботи проєкту потрібно встановити Python версії **3.10 або новішої**.

Перевірити встановлену версію можна командою:

```bash
python --version
```

або:

```bash
python3 --version
```

---

### 2. Клонування репозиторію

```bash
git clone https://github.com/Volllchok/Dyplom.git
cd your-repository
```

### 3. Створення віртуального середовища

```bash
python -m venv venv
```

Активація середовища у Windows:

```bash
venv\Scripts\activate
```

Активація середовища у macOS / Linux:

```bash
source venv/bin/activate
```

---

### 4. Встановлення залежностей

```bash
pip install numpy pandas matplotlib seaborn scipy scikit-learn openpyxl
```

За потреби можна створити файл `requirements.txt`:

```text
numpy
pandas
matplotlib
seaborn
scipy
scikit-learn
openpyxl
```

І встановити залежності командою:

```bash
pip install -r requirements.txt
```

---

### 5. Підготовка вхідних файлів

У папку `data/` потрібно додати Excel-файли:

```text
data/data_slider_1.xlsx
data/Err_data_slider_1.xlsx
```

У коді потрібно вказати правильний шлях до файлу з даними:

```python
file_path = 'data/data_slider_1.xlsx'
sheet_name = 'data'
```

Для аналізу похибок потрібно вказати шлях до файлу:

```python
plot_error_analysis('data/Err_data_slider_1.xlsx')
```

---

### 6. Запуск програми

```bash
python main.py
```

Після запуску програма будує основні графіки:

- просторову карту сигналу сенсора `S1`;
- просторову карту сигналу сенсора `S2`;
- просторову карту сигналу сенсора `S6`;
- карту домінуючого сенсора;
- PCA-представлення сенсорних сигналів;
- кореляційну матрицю;
- графіки аналізу похибок прогнозування координат.

## 🖱️ Інструкція для користувача

1. Відкрити проєкт у середовищі розробки, наприклад Visual Studio Code або PyCharm.
2. Переконатися, що файли `data_slider_1.xlsx` та `Err_data_slider_1.xlsx` розміщені у правильній папці.
3. Перевірити шлях до файлів у змінних `file_path` та `error_file`.
4. Запустити файл `main.py`.
5. Переглянути побудовані графіки у вікнах Matplotlib.
6. За потреби змінити номер сенсора у виклику функції:

```python
plot_sensor_map(sensor_id=0)
```

де:

- `sensor_id=0` відповідає сенсору `S1`;
- `sensor_id=1` відповідає сенсору `S2`;
- `sensor_id=5` відповідає сенсору `S6`.

## 🧪 Проблеми і рішення

| Проблема | Можлива причина | Рішення |
|---------|-----------------|---------|
| `FileNotFoundError` | Неправильно вказаний шлях до Excel-файлу | Перевірити шлях у змінній `file_path` |
| `ModuleNotFoundError` | Не встановлено потрібну бібліотеку | Виконати `pip install numpy pandas matplotlib seaborn scipy scikit-learn openpyxl` |
| Не відкривається Excel-файл | Відсутня бібліотека `openpyxl` | Виконати `pip install openpyxl` |
| Графіки не відображаються | Проблема з backend Matplotlib або запуском у середовищі | Запустити програму локально через VS Code, PyCharm або термінал |
| На тепловій карті є порожні ділянки | Інтерполяція не змогла обчислити значення для всіх точок | Перевірити щільність вхідних даних або змінити метод інтерполяції |
| Помилка у PCA | У даних є порожні або некоректні значення | Перевірити Excel-файл на пропуски та нечислові значення |

## ✅ Результат роботи програми

Результатом роботи програми є набір графіків, які дозволяють оцінити поведінку лінійного сенсорного слайдера. Просторові карти показують зони чутливості окремих сенсорів, карта домінуючого сенсора демонструє розподіл зон відповідальності каналів, PCA-графік відображає структуру багатовимірних сигналів, а кореляційна матриця показує зв’язки між сенсорами. Аналіз похибок дозволяє визначити ділянки, де точність прогнозування координати є нижчою, що може бути використано для подальшої калібровки сенсорного пристрою.
