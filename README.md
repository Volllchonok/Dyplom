# 📘 Візуалізація даних сенсорного слайдера

> Програмний проєкт для аналізу та візуалізації даних лінійного сенсорного слайдера. Програма зчитує експериментальні дані з Excel-файлу, будує просторові карти сигналів сенсорів, карту домінуючого сенсора, PCA-представлення, кореляційну матрицю та графіки похибок прогнозування координат.

## 📌 Загальна інформація

- **Тип проєкту**: локальний Python-скрипт для аналізу сенсорних даних
- **Мова програмування**: Python
- **Формат вхідних даних**: Excel-файли `.xlsx`
- **Основні бібліотеки**: NumPy, Pandas, Matplotlib, Seaborn, SciPy, scikit-learn
- **Призначення**: візуалізація, контроль та діагностика роботи лінійного сенсорного слайдера

Проєкт складається з одного основного Python-файлу. У цьому файлі реалізовано зчитування даних, обробку сигналів, побудову графіків та аналіз похибок. Окрема структура з frontend, backend або API у проєкті не використовується.

---

## 🧠 Опис функціоналу

Програма виконує такі дії:

- 📥 зчитує експериментальні дані з файлу `data_slider_1.xlsx`;
- 📊 будує просторові карти сигналів окремих сенсорів;
- 🟦 визначає домінуючий сенсор у кожній точці дотику;
- 📉 виконує PCA-аналіз восьми сенсорних сигналів;
- 🔗 будує кореляційну матрицю між сенсорними каналами;
- 📍 аналізує похибки прогнозування координат точки дотику;
- 🌡️ будує теплову карту просторового розподілу похибок;
- 📈 будує гістограму розподілу похибок.

---

## 🧱 Опис основного файлу та функцій

| Файл / Функція | Призначення |
|---------------|-------------|
| `main.py` | Основний файл програми, у якому міститься весь код для зчитування даних, аналізу та побудови графіків |
| `data_slider_1.xlsx` | Excel-файл із координатами точки дотику `X`, `Y` та сигналами восьми сенсорів `S1–S8` |
| `Err_data_slider_1.xlsx` | Excel-файл із даними для аналізу похибок прогнозування координат |
| `plot_sensor_map(sensor_id)` | Будує просторову карту сигналу вибраного сенсора |
| `plot_dominant_sensor_map()` | Визначає сенсор із максимальним сигналом у кожній точці та будує карту домінуючого сенсора |
| `plot_pca()` | Виконує PCA-аналіз і відображає восьмивимірні сенсорні сигнали у двовимірному просторі |
| `plot_correlation_matrix()` | Будує кореляційну матрицю між сенсорними каналами |
| `plot_error_analysis(error_file)` | Аналізує похибки прогнозування координат і будує графік, гістограму та теплову карту похибок |

## 📂 Структура проєкту

Проєкт не має складної файлової структури, оскільки весь програмний код розміщений в одному Python-файлі.

```text
main.py
README.md
data_slider_1.xlsx
Err_data_slider_1.xlsx
```

Файл `main.py` містить усі функції для роботи з даними. Файли `data_slider_1.xlsx` та `Err_data_slider_1.xlsx` використовуються як вхідні дані. Якщо Excel-файли зберігаються в іншій папці, потрібно змінити шлях до них у коді.

## ▶️ Як запустити проєкт з нуля

### 1. Встановити Python

Для роботи програми потрібно встановити Python версії **3.10 або новішої**.

Перевірити версію Python можна командою:

```bash
python --version
```

або:

```bash
python3 --version
```

---

### 2. Завантажити або скопіювати файли проєкту

Потрібно мати основний файл програми, наприклад:

```text
main.py
```

Також потрібно підготувати Excel-файл із даними сенсорного слайдера:

```text
data_slider_1.xlsx
```

Для аналізу похибок додатково використовується файл:

```text
Err_data_slider_1.xlsx
```

---

### 3. Створити віртуальне середовище

```bash
python -m venv venv
```

Активація у Windows:

```bash
venv\Scripts\activate
```

Активація у macOS / Linux:

```bash
source venv/bin/activate
```

---

### 4. Встановити залежності

```bash
pip install numpy pandas matplotlib seaborn scipy scikit-learn openpyxl
```

За бажанням можна створити файл `requirements.txt` з таким вмістом:

```text
numpy
pandas
matplotlib
seaborn
scipy
scikit-learn
openpyxl
```

Після цього залежності можна встановити командою:

```bash
pip install -r requirements.txt
```

---

### 5. Налаштувати шлях до файлу з даними

У коді потрібно вказати правильний шлях до Excel-файлу з експериментальними даними:

```python
file_path = 'C:/Слайдер_13_11_25/data_slider_1.xlsx'
sheet_name = 'data'
```

Якщо файл лежить поруч із `main.py`, шлях можна змінити так:

```python
file_path = 'data_slider_1.xlsx'
```

Якщо файл лежить у папці `data`, шлях можна вказати так:

```python
file_path = 'data/data_slider_1.xlsx'
```

Для аналізу похибок потрібно розкоментувати останній рядок і вказати шлях до файлу з похибками:

```python
plot_error_analysis('C:/Слайдер_13_11_25/Err_data_slider_1.xlsx')
```

або, якщо файл лежить поруч із програмою:

```python
plot_error_analysis('Err_data_slider_1.xlsx')
```

---

### 6. Запустити програму

```bash
python main.py
```

Після запуску програма побудує основні графіки:

- просторову карту сигналу сенсора `S1`;
- просторову карту сигналу сенсора `S2`;
- просторову карту сигналу сенсора `S6`;
- карту домінуючого сенсора;
- PCA-представлення сенсорних сигналів;
- кореляційну матрицю.

Для побудови графіків аналізу похибок потрібно розкоментувати рядок:

```python
# plot_error_analysis('C:/Слайдер_13_11_25/Err_data_slider_1.xlsx')
```

і замінити шлях на актуальний.


- `sensor_id=0` відповідає сенсору `S1`;

- `sensor_id=5` відповідає сенсору `S6`.

## 🧪 Проблеми і рішення

| Проблема | Можлива причина | Рішення |
|---------|-----------------|---------|
| `FileNotFoundError` | Неправильно вказаний шлях до Excel-файлу | Перевірити значення змінної `file_path` |
| `ModuleNotFoundError` | Не встановлено потрібну бібліотеку | Виконати команду `pip install numpy pandas matplotlib seaborn scipy scikit-learn openpyxl` |
| Не відкривається Excel-файл | Не встановлена бібліотека `openpyxl` | Виконати команду `pip install openpyxl` |
| Графіки не відображаються | Програма запускається у середовищі без підтримки графічних вікон | Запустити програму локально у VS Code, PyCharm або через термінал |
| На тепловій карті є порожні ділянки | Інтерполяція не змогла побудувати значення для всіх точок | Перевірити щільність вхідних даних або змінити метод інтерполяції |
| Помилка під час PCA | У даних є порожні або нечислові значення | Перевірити Excel-файл на пропуски та некоректні значення |

## ✅ Результат роботи програми

Результатом роботи програми є набір графіків, які дозволяють оцінити поведінку лінійного сенсорного слайдера.
